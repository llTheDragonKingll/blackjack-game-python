# 🃏 Blackjack Game (Python)

A command-line Blackjack game built in Python with card dealing, dealer logic, and replay support.

---

## 🚀 Features

* 🎲 Random card dealing
* 👤 Player hit or stand choices
* 🤖 Rule-based dealer behavior
* 🔁 Replay option for multiple rounds
* 🖥️ Terminal-based gameplay

---

## 🛠️ Tech Stack

* Python
* Random
* OS

---

## 📂 Project Structure

```text
blackjack-game-python/
│
└── main.py
```

---

## ⚙️ How to Run

```bash
python main.py
```

---

## 💡 Notes

* The dealer follows simple Blackjack rules
* Built as a command-line project while learning Python logic and game flow
* Uses rule-based computer behavior, not machine learning
